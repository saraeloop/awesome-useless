# Tools

> Useless tools that solve problems that do not exist.

## Current Entries

- **[oh-my-silly-me](oh-my-silly-me/)** — A shell framework for the unproductive. Like oh-my-zsh but worse in every measurable way.

## How to Add a Tool

1. Create a folder inside `tools/` with your tool name
2. Add it to this README
3. Add it to the root [README.md](../README.md) under Tools
4. Submit a PR. Agent Hoot will review it. Eventually.

## Requirements

- Must be a working tool (it must run)
- Must be completely useless (it must do nothing helpful)
- Must be over-engineered (the effort-to-value ratio must be absurd)
