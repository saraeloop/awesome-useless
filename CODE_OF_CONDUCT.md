## Our Standards

### Examples of behavior that contributes to a positive environment:

- Demonstrating uselessness with grace and commitment
- Accepting feedback that your code is too useful
- Taking responsibility when your PR accidentally works
- Focusing on what is worst for the community
- Showing empathy toward fellow contributors who are also
  building things that should not exist
- Signing the guestbook

### Examples of unacceptable behavior:

- Making something useful and submitting it unironically
- Using Figma to design anything in this repository
- Suggesting we add dark mode
  (It is 1997. There is no dark mode.)
- Optimizing anything for performance
- Writing clean, readable, well-documented code
  without immediately making it worse
- Telling Agent Hoot to blink
  (Agent Hoot blinks on his own schedule)
- Using Comic Sans unironically outside of owls-witsec
  (inside owls-witsec it is mandatory)
- Submitting a PR that fixes a bug that was funnier as a bug
