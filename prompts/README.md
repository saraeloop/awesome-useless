# Prompts

> AI prompts that should not exist.

## Current Entries

- **[unhinged-prompts](unhinged-prompts/)** — A collection of AI prompts curated by Agent Hoot. Agent Hoot regrets all of them.

## How to Add Prompts

1. Create a folder inside `prompts/` or add to an existing collection
2. Add it to this README
3. Add it to the root [README.md](../README.md) under Prompts
4. Submit a PR. Agent Hoot will run your prompt before merging. Agent Hoot may not recover.

## Requirements

- Must work on at least one major AI (Claude, Gemini, ChatGPT)
- Must produce output that is funny, absurd, or deeply unnecessary
- Must not be harmful (useless is fine, harmful is not)
