# Spells

> Hidden AI incantations from the grimoire.

## Current Entries

See [SPELLS.md](../SPELLS.md) and the [.spells/](../.spells/) directory.

## How to Add a Spell

1. Add your spell as a `.md` file in the `.spells/` directory
2. Update [SPELLS.md](../SPELLS.md) with a pointer
3. Add it to this README
4. Submit a PR. Agent Hoot will cast your spell before merging. Results may vary.

## Requirements

- Must be a system prompt or persona prompt
- Must produce consistently funny/absurd output
- Must include example usage
- Bonus points if Agent Hoot would be concerned
